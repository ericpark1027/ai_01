랜딩 페이지 구축 가이드 Landing Page Build Guide

초보자도 쉽게 따라 할 수 있는 웹사이트 제작 및 배포 과정을 소개합니다. "Apocalypse Girl" 영상을 활용한 멋진 페이지를 만들어보세요. Introducing the process of building and deploying a website that beginners can easily follow. Create a stunning page using the "Apocalypse Girl" video.

Project Name: Apocalypse Girl

1

HTML 구조 설계 Designing HTML Structure

가장 먼저 웹사이트의 뼈대를 만듭니다. semantic 태그(header, main, footer)를 사용하여 검색 엔진이 이해하기 쉬운 구조로 코드를 작성하세요.

First, create the skeleton of your website. Use semantic tags (header, main, footer) to write code that is easy for search engines to understand.

2

비디오 요소 삽입 Inserting Video Elements

<video> 태그를 사용하여 준비된 소스를 연결합니다. GitHub에 파일을 업로드했다면, 'raw' URL을 사용하여 브라우저에서 직접 재생될 수 있도록 설정해야 합니다.

Connect your prepared source using the <video> tag. If you uploaded files to GitHub, you must use the 'raw' URL so that it can be played directly in the browser.

3

다크 모드 & 다국어 설정 Dark Mode & Multi-language

사용자의 가독성을 위해 테마 전환 버튼과 언어 전환 기능을 추가합니다. Tailwind CSS의 'dark' 클래스와 간단한 JavaScript 함수로 구현할 수 있습니다.

Add a theme toggle and language switch for user readability. This can be implemented with Tailwind CSS's 'dark' class and simple JavaScript functions.

4

GitHub Pages 배포 Deploy to GitHub Pages

마지막으로 `index.html` 파일을 저장한 뒤 GitHub Repository에 푸시하세요. 설정(Settings)의 Pages 탭에서 브랜치를 선택하면 전 세계 누구나 접속 가능한 URL이 생성됩니다.

Finally, save your `index.html` file and push it to your GitHub Repository. Select the branch in the Pages tab of Settings to generate a URL accessible to anyone in the world.
